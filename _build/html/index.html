Chapi

Chapi is the alternative name for coot_headless_api and is the Pythonic interface to libcootapi. It is a clear and consistent and easy to use high level interface to the functions of Coot. On creating a new molecule, a molecule index will be returned. Molecules are referred to by this index and using the functions of molecules_container_t. This is unlike many other functions of Python modules, which return a Python representation of the data.
